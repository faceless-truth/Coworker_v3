# MC&S Individual Tax Return Methodology

## Purpose

This methodology document sets out the standardised approach for preparing, reviewing, and finalising individual income tax returns at MC&S. The objective is to ensure:

* Consistency across all preparers.
* Comprehensive identification of deductible claims.
* Accurate comparison to prior-year returns.
* Compliance with ITAA 1997, ATO rulings, and substantiation requirements.
* Appropriate risk management and audit awareness.
* High-quality client communication and documentation.

---

# 1. Core Tax Return Preparation Principles

Every return should be prepared using the following guiding principles:

1. Compare current-year return against prior-year return.
2. Question significant movements, omissions, or new claims.
3. Ensure all deductions have a nexus to assessable income.
4. Verify substantiation requirements are met.
5. Distinguish between deductible, capital, and private expenditure.
6. Apply occupation-specific deduction knowledge.
7. Review all ATO prefill data carefully.
8. Assess audit risk before lodgment.
9. Document assumptions and client explanations.
10. Ensure consistency between workpapers, schedules, and lodged return.

---

# 2. Standard Tax Return Workflow

## Step 1 – Initial Review and Client Information

Review:

* Prior-year tax return.

* Current-year client questionnaire.

* Investment activities.
* New assets or disposals.
* Work-from-home arrangements.
* Motor vehicle usage.
* Cryptocurrency activity.

# 3. Mandatory Prior-Year Comparison Procedures

## Prior-Year Comparison is Mandatory

Always compare the current-year return against the previous lodged return.

## Key Items to Compare

### Income

Review for:

* Missing employers.
* Additional income streams.
* Investment income changes.
* Reduced or increased wages.
* Foreign income.
* Director fees.
* Government payments.
* PSI indicators.
* Business cessation or commencement.

### Deductions

Compare all major deduction categories:

| Deduction Category    | Review Required                       |
| --------------------- | ------------------------------------- |
| Motor vehicle         | Compare km, method, % usage           |
| Work-related travel   | Compare frequency and substantiation  |
| Self-education        | Check course continuity and relevance |
| Uniform/laundry       | Compare consistency                   |
| Home office           | Compare hours and methodology         |
| Phone/internet        | Compare apportionment                 |
| Tools/equipment       | Check additions/disposals             |
| Donations             | Verify receipts                       |
| Tax agent fees        | Compare reasonableness                |
| Rental deductions     | Compare expense trends                |
| Interest expenses     | Compare to loan balances              |
| Repairs & maintenance | Determine repair vs improvement       |
| Depreciation claims   | Verify continuity                     |

## Mandatory Questions for Variances

If any deduction materially changes from prior year:

* Ask client why.
* Document explanation.
* Obtain additional substantiation if required.
* Assess audit risk.

Examples:

| Scenario                       | Required Action                         |
| ------------------------------ | --------------------------------------- |
| Car claim doubles              | Request updated logbook and explanation |
| No WFH claim this year         | Confirm no omission                     |
| Repairs significantly increase | Determine capital vs repair             |
| No rental interest claimed     | Verify loan refinance or offset usage   |
| Large drop in income           | Confirm employment change               |

# 4. Occupation-Specific Deduction Review

## Mandatory Occupation Analysis

Every return must include consideration of occupation-specific deductions.

Preparers should:

* Identify the client occupation.
* Refer to ATO occupation guides where available.
* Consider common and less common deductions.
* Exclude private or non-deductible items.

---

### Employees

* Work-related phone and internet.
* Tools and equipment.
* Reference materials.
* PPE and safety equipment.
* Uniforms with logos.

### Teachers

Additional review:

### Medical Professionals

* Scrubs/protective gear.

# 5. High-Risk Deduction Areas

The following deductions require heightened review.

## Motor Vehicle Claims

Review:

* Logbook validity.
* Odometer records.
* Business-use percentage.
* Employer reimbursements.
* Novated lease arrangements.
* Private travel exclusion.

Red Flags:

* Exactly 5,000 km every year.
* Large increases without explanation.
* Claim inconsistent with occupation.
* Claim exceeds practical work travel.

---

## Work From Home Claims

Review:

* Diary or records.
* Occupation suitability.
* Fixed rate vs actual method.
* Double-dipping risks.
* Apportionment methodology.

Red Flags:

* Excessive hours.
* No evidence.
* Client works predominantly onsite.

---

## Self-Education

Review:

* Nexus to current income.
* Private benefit.
* New employment pathway.
* Travel substantiation.

Red Flags:

* Course unrelated to current role.
* Qualification for career change.

---

## Rental Property Repairs

Review:

* Initial repairs.
* Improvements.
* Replacement of entirety.
* Capital works treatment.
* Insurance recoveries.

Red Flags:

* Large one-off repair claims.
* Renovation expenditure coded as repairs.
* Property recently purchased.

---

# 6. Rental Property Methodology

## Mandatory Rental Property Review

For every rental property, obtain:

* Purchase date.
* Settlement statement.
* Ownership percentages.
* Loan statements.
* Property manager annual statement.
* Depreciation schedule.
* Details of private use.
* Details of redraws or refinances.

---

## Rental Property Checklist

### Deductible Items

Review:

* Interest.
* Council rates.
* Water rates.
* Insurance.
* Agent fees.
* Repairs.
* Pest control.
* Cleaning.
* Gardening.
* Body corporate.
* Land tax.
* Borrowing costs.
* Quantity surveyor fees.
* Depreciation.
* Capital works deductions.

### Non-Deductible / Capital Items

Review:

* Initial repairs.
* Renovations.
* Improvements.
* Travel to inspect residential property.
* Private use periods.

---

## Loan Review Procedures

Always confirm:

* Whether redraws occurred.
* Whether offset accounts were used privately.
* Whether refinances included private debt.
* Whether loan purpose changed.

Interest deductibility must follow the use of borrowed funds.

# 10. ATO Audit Risk Review

## Risk Categories

### Low Risk

Examples:

* Standard union fees.
* Professional memberships.
* Minor stationery.
* Substantiated uniform expenses.

### Medium Risk

Examples:

* Work from home claims.
* Moderate vehicle claims.
* Self-education.
* Rental repairs.

### High Risk

Examples:

* Large vehicle deductions.
* Large work from home claims.
* Unusual deductions.
* Significant repair claims.
* Aggressive apportionments.
* Missing substantiation.

---

## Mandatory Risk Notes

Preparers must document:

* Risk rating.
* Why the claim is supportable.
* What substantiation was reviewed.
* Any limitations or assumptions.

# 11. Final Review Procedures

Before lodgment, confirm:

## Income

* All prefill data included.
* Investment income included.
* Foreign income reviewed.
* Government payments included.
* No duplicated income.

## Deductions

* Prior-year comparison completed.
* Claims reasonable for occupation.
* Workpapers complete.
* Substantiation noted.
* Capital/private expenses excluded.

## Rental Properties

* Interest reconciled.
* Depreciation updated.
* Repairs reviewed.
* Ownership percentages correct.
* Private use adjusted.

## CGT

* Discounts applied correctly.
* Capital losses utilised correctly.
* Cost base supported.

## Compliance

* Medicare levy reviewed.
* MLS reviewed.
* HECS/HELP reviewed.
* Division 293 considered.
* PAYGI reviewed.

# 12. Client Communication Standards

Preparers should:

* Explain significant changes from prior year.
* Highlight high-risk deductions.
* Advise substantiation retention requirements.
* Explain ATO review risk where applicable.
* Confirm unresolved matters before lodgment.

# 13. Common Errors to Avoid

## Frequently Identified Issues

* Claiming home-to-work travel.
* Claiming conventional clothing.
* Incorrect rental repair treatment.
* Missing foreign income.
* Double-claiming work-from-home expenses.
* Incorrect interest apportionment.
* Claiming private expenses.
* Failing to compare prior year.
* Missing capital gains events.
* Incorrect ownership splits.

# 16. MC&S Standard Review Philosophy

The return preparer should adopt the following mindset:

1. Assume omissions are possible.
2. Compare every major item to prior year.
3. Question unusual movements.
4. Verify deductibility under ITAA97.
5. Focus on substantiation.
6. Distinguish repairs from capital.
7. Apply commercial reasonableness.
8. Document everything material.
9. Assess ATO audit risk.
10. Ensure the file can be defended during review or audit.

---

# 17. Key Internal Rule

## Mandatory Prior-Year Comparison Rule

No individual return is considered complete unless:

* The prior-year return has been reviewed.
* Major movements have been explained.
* Missing prior-year deductions have been investigated.
* Material inconsistencies have been documented.

This is a mandatory MC&S review procedure.
